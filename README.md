# DashBoardsFE (dbs)

_Project conventions_:
* plus ( + ) before variable used for convert String to Number (Ex. +'1' === 1)
* double exclamation ( !! ) before variable used for convert String to Boolean (Ex. !!1 === true, !!0 === false)
* setTimeout(function, 0) use for wait a tick first to avoid one-time devMode unidirectional-data-flow-violation error
